## 代码演示

```tsx
const Index = () => {
  return <ToolCard gridCol={5} maxWidth={686} iconSize={70} />;
};
```

## Props

|   参数   |  类型  | 默认值 |      说明      |
| :------: | :----: | :----: | :------------: |
| gridCol  | number |        |  分为多少栏目  |
| maxWidth | number |        | 卡片的最大宽度 |
| iconSize | number |        |    图标大小    |
